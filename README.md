## 追加ライブラリ
```
npm install classnames
npm install dayjs@1.11.10
npm install nextjs-basic-auth-middleware@3.1.0
```

## Next.jsの開発モードでサーバーを起動
```
npm run dev

- ローカルサーバー
http://localhost:3000/
```
